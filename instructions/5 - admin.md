# Foundry – Admin Interface

## Objectif
Un back-office simple, rapide et fonctionnel.
Pas de sur-design, pas de complexité inutile.

## Sections principales

### Sites
- Liste des sites
- Création d’un site
- Modification :
  - nom
  - thème
  - statut

### Domaines
- Ajout de domaines
- Définir le domaine primaire
- Activer/désactiver les redirections

### Contenu
- Pages
- Articles
- Statut draft / published
- Édition HTML contrôlée

### Taxonomies
- Catégories
- Tags

### Media
- Upload
- Association au contenu

## UX attendue
- Chargements rapides
- Peu de clics
- Actions claires
- Pas d’édition WYSIWYG complexe au départ

## Sécurité
- Accès admin protégé
- Pas d’exposition publique des routes admin
